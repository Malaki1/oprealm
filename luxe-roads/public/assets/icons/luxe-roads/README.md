# Luxe Roads icon crops

These PNG files are cropped from the supplied Luxe Roads category and travel icon sheets.

- `categories/` contains the 8 browse-by-style icon crops.
- `travel-set/` contains the travel, support, assurance, and add-on icon crops used by process steps, trust badges, and add-on cards.

The source sheets are visual references only; the live page uses these individual cropped assets.

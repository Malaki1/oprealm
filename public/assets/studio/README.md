# OPREALM Studio Launcher Assets

This folder holds the reusable assets for the simplified category-first creator launcher.

- `backgrounds/` full-screen world art and future parallax layers.
- `cards/` large creation category artwork for Games, Stories, Music and Movies.
- `thumbnails/` recent project previews.
- `orbit/` mascot states for helper, loading and celebration moments.
- `icons/` neon category badges.
- `floating/` transparent decorative world objects.
- `fx/` particles, sparkles, fog and streak overlays.
- `glows/` reserved for future glow overlays if CSS gradients are not enough.

Typography stays in HTML/CSS so the artwork remains responsive, accessible and reusable.

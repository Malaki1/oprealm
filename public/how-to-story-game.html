Creator walkthrough

Follow the story game flow.

01

Create a hero from a drawing or idea.

Open the Character tab. Choose whether to upload a drawing or use the prompt builder. Add the hero name, what they are, their personality, and the visual style.

  • Use clear character details like hair, eyes, outfit, ears, species and colors.
  • Pick the same visual style you want scene cards to use later.
  • Save the hero before moving to the next step.

02

Choose the title, genre, audience and vibe.

Move to Game Cover. Add the game title, audience, genre and short direction. OPREALM turns those details into a cover prompt for approval.

  • Keep the title short and memorable.
  • Pick an audience so the story tone stays age-appropriate.
  • Use the cover to set the whole game mood.

03

Generate a branching story script.

Use the Story Engine to create a scene-by-scene plan with choices, plot twists and different endings. This becomes the backbone of the playable story.

  • Happy, sad, cliff-hanger and satisfying endings can all exist in one map.
  • Each choice should lead somewhere meaningful.
  • You can edit the story direction before making scene images.

04

Create 16:9 scene cards one by one.

Use Scene Cards to generate each visual story moment. Approve the image, edit it with a prompt if needed, then create the next scene in order.

  • Keep the same saved hero and visual style locked in.
  • Use previous scene cards to keep the look consistent.
  • Enlarge images before approving if you need to inspect details.

05

Add the Banner UI text layer.

Pick a banner style, font, color, text size and alignment. Drag the text box into the right area and make sure the question or narration stays readable.

  • Short text works best on scene cards.
  • Use white text on dark images and dark text on light banners.
  • Resize banner and button elements until the card feels balanced.
Banner UI

06

Connect the choices in the Story Map.

Open Story Map. Use the dots on each scene card to connect routes so players can see how each choice leads to another moment or ending.

  • Make sure every button has a destination.
  • Keep the map readable with clear scene names.
  • Test the story path from beginning to ending.

Final check

Preview, approve and publish safely.

Before publishing, play through the story, check each scene image, confirm the text is readable, and make sure every choice is kind, age-appropriate and safe.
